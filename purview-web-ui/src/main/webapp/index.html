1. Find compression errors

By recompressing JPEG images, it is possible to detect regions in an image that are of a different quality relative to its surroundings.

Purview is able to detect such quality differences via a sophisticated image recompression algorithm.

2. Highlight scaled regions

Purview lets you analyse your image to find areas that have been scaled.

Currently supported scaling algorithms:

  1. Nearest-neighbor interpolation
  2. Bilinear resampling
  3. Bicubic interpolation

3. Detect cloned regions

You thought that it was a good idea to cover something up using the Clone stamp tool?

Well, you better think again! Purview is able to detect cloned regions in an image!

4. Analyse metadata

Most image editors leave behind various traces in an image's meta data.

Via a simple metadata recognition database, Purview is able to detect images that have been edited by specific editors.

5. Extensions

Doesn't Purview currently detect an image forgery technique that you know of?

No problem, Purview is completely modular and can easily be extended with new analysers!
